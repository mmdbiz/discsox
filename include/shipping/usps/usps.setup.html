QuikStore 3.0 USPS XML Rate Calculator Plug-in Setup

Registering at United States Postal Service

The first thing you need to do is register with USPS Online Tools so you can get a username. The password is no longer required.

Go here to register https://secure.shippingapis.com/registration/

Once you have registered, you will get an email with your "TEST" username and password. We will be using this during the test phase of the plug-in.

Plug-in Installation Instructions

To use USPS:

  1. Login in to your control panel and select Calculations from the top, then Shipping Rates/General on the left.
  2. Select true for "Use Shipping Plugin".
  3. Enter: usps.xml.inc in the "Shipping Plugin Name" field and click Update Record.
  4. Download and then open the usps.config.php file in a text editor and change the following variables:

    usps_user_id=YOUR USPS Username
    usps_origin_postal_code=YOUR zip code

  5. You can also set any other variables as you wish in the usps.config.php file.
    Each variable in the usps.config.php file has been commented as to its' purpose/settings.

Testing The USPS Plug-in

To make sure the plug-in is working correctly, we need to run the test script:

In a browser, enter the URL to the uspstest.cgi script:

http://www.YOURSITE.com/usps.test.php

This should return a list of tests calls to the USPS server. If all of these values got returned correctly, you are ready to go live...

NOTE: USPS will NOT let you go live until these tests have been completed.

Going Live with USPS

Once you have successful tested against the test server and feel you are ready to send requests to the "live" production server you will need to contact the Internet Customer Care Center (ICCC) by telephone or email and they will activate your user ID and password so that you have access to the production server at production.shippingapis.com.

The ICCC is manned from 7:00AM to 11:00PM Eastern Time.
E-mail: icustomercare@usps.com
Telephone: 1-800-344-7779 (7:00AM to 11:00PM ET)

 

Once a shipping plug-in is turned on, and has been successful tested, go to the edit product screen for one of your products. You will now see a link below the weight field that will help you to define a package for this product. Use this to define a package for at least one product for testing. Now add that product to the cart and go to checkout. This will test to make sure USPS is giving you live rates when you go to checkout.
